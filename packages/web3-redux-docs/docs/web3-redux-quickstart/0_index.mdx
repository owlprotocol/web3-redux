# Quickstart

This quickstart is meant to get you up and running with Web3-Redux and start using some of its basic features. The guide will cover the following:

-   [Install](./1_install.md): Install Web3-Redux and peer dependencies.
-   [Configure Store](./2_add_store.md): Initiallize the Redux Store and setup the React Context Provider.
-   [Configure Network](./3_add_network.mdx): Add a network configuration with an RPC connection. (eg. Ethereum Mainnet)
-   [Configure Contract](./4_add_contract.md): Add a contract with its ABI. (eg. ERC20 Token)
-   [Contract Call](./5_contract_call.md): Make a contract method call. (eg. `balanceOf(account)`)
-   [Contract Events](./6_contract_event.md): Get past and sync new contract events. (eg. `Transfer` events)
-   [Conclusion](./7_conclusion.md): Where to go from here for more advanced usage.
