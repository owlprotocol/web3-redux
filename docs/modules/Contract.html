callBatched

callBatched: ActionCreatorWithPayload<CallBatchedActionInput, string> = ...
  • Defined in packages/web3-redux/src/contract/actions/callBatched.ts:30

Optimally batched call requests. Requests are grouped by network and batched with web3.BatchRequest().

see

https://web3js.readthedocs.io/en/v1.2.0/web3-eth.html#batchrequest

Calls will be batched busing Multicall if:

  • network has a Multicall contract
  • from == undefined
  • defaultBlock == 'latest' || defaultBlock == undefined
see

https://github.com/makerdao/multicall
